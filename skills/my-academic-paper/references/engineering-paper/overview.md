# Engineering Paper Overview

This folder contains the route-specific rules for `工科学术论文`.

## File Map

- `workflow.md`: overall workflow, section order, start gate, and separate-file requirements
- `abstract.md`: abstract writing rules
- `literature_review.md`: literature review rules
- `introduction.md`: introduction-writing rules
- `method.md`: method-writing rules
- `results.md`: results-writing rules
- `discussion.md`: discussion-writing rules
- `conclusion.md`: conclusion-writing rules
- `future_work.md`: future-work rules
- `style.md`: shared structural, citation, wording, and formatting rules

## Usage

When the route is `工科学术论文`:

1. Read `workflow.md` first
2. Read `style.md` as the global constraints
3. Read the section-specific file before drafting that section
4. Keep each major part as a separate file artifact during drafting
