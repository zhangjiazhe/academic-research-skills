# Engineering Paper Results Rules

1. Results must combine quantitative findings with concise qualitative interpretation; purely qualitative description is not acceptable.
2. Every experiment, table, figure, and metric result must be analyzed, even if the analysis is brief.
3. When a later claim relies on an earlier finding, explicitly explain how the reported result supports that claim.
4. Avoid long descriptive paragraphs that do not report data, metrics, or evidence.
