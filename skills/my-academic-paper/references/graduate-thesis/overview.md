# Graduate Thesis Overview

This folder contains the route-specific rules for `毕业论文`.

## File Map

- `workflow.md`: overall workflow, chapter order, body-first drafting order, and conclusion/introduction sequencing
- `style.md`: whole-thesis shared wording, reference, formatting, figure/table/formula, and citation-placement rules
- `chapter_intro.md`: `章前言` rules
- `section_preface.md`: `节前言` rules
- `method.md`: method-writing rules
- `results.md`: results-writing rules
- `discussion.md`: discussion-writing rules
- `chapter_summary.md`: `章小结` rules
- `conclusion.md`: `主要结论` rules
- `innovation.md`: `主要创新` rules
- `future_work.md`: `不足与未来展望` rules
- `introduction.md`: final Introduction rules

## Usage

When the route is `毕业论文`:

1. Read `workflow.md` first
2. Read `style.md` as the whole-thesis shared constraints
3. Read the section-specific file before drafting that section
4. Keep the chapter structure clear:
   - `章前言`
   - 正文
   - `章小结`
5. In actual drafting order for regular chapters, write:
   - 正文 first
   - then `章小结`
   - then `章前言`
